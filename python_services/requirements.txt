# Installation Notes:
# 1. Install Python 3.8+ from python.org
# 2. Create virtual environment: python -m venv venv
# 3. Activate virtual environment:
#    - Windows: venv\Scripts\activate
#    - Mac/Linux: source venv/bin/activate
# 4. Install dependencies: pip install -r requirements.txt
# 5. If MediaPipe fails on older systems, try: pip install mediapipe --no-deps
#    then manually install numpy and opencv-python

# Platform-specific notes:
# Windows: Ensure Visual C++ Redistributable is installed
# macOS: May require Xcode command line tools: xcode-select --install
# Linux: May require additional packages: sudo apt-get install python3-dev cmake

# Troubleshooting:
# - If cv2 import fails: pip uninstall opencv-python && pip install opencv-python
# - If MediaPipe fails: Try pip install mediapipe --user
# - For ARM processors (M1 Mac): Some packages may need conda instead of pip
